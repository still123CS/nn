# rainy_fusion_3d_det
基于多传感器融合的自动驾驶汽车雨天条件下3D物体检测模拟项目

## 项目简介
本项目针对自动驾驶汽车在雨天条件下感知性能下降的痛点，基于多传感器融合技术，构建了一套雨天场景下的3D物体检测模拟系统。

项目通过融合相机、激光雷达等多源传感器数据，有效克服雨天环境中的雨雾遮挡、光线反射、点云噪声等干扰，提升自动驾驶车辆在复杂天气条件下对行人、车辆、障碍物等目标的检测精度与鲁棒性，为雨天环境下的安全驾驶提供感知层面的技术支撑。

## 主要功能
- 雨天场景多传感器数据模拟与预处理
- 相机-激光雷达数据时空同步与特征级融合
- 融合型3D物体检测模型训练与推理
- 雨天干扰下检测性能对比与可视化分析
