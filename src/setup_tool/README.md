# Setup Tool - 进度显示演示工具

## 模块概述

本模块演示了为项目根目录的 `setup.bat` 添加的实时进度显示功能。

## 功能演示

运行本模块可查看 `setup.bat` 的进度显示效果，无需实际执行安装操作。

### 进度显示特性

- **实时进度条**：50 字符宽度的可视化进度条
- **百分比显示**：清晰的完成百分比（如 63%）
- **步骤编号**：当前步骤 / 总步骤（如 Step 7/11）
- **计时功能**：
  - 已用时间（Elapsed）
  - 预估剩余时间（ETA）
- **标准化状态输出**：
  - `[OK]` - 步骤成功
  - `[SKIP]` - 步骤跳过
  - `[DOWNLOAD]` - 下载文件
  - `[ERROR]` - 步骤失败
- **统计汇总**：完成后的完整统计信息

## 运行方法

### 方式一：演示模式（推荐）

在项目根目录运行：

```bash
python src/setup_tool/main.py
```

### 方式二：实际安装

在项目根目录运行：

```batch
setup.bat
```

## setup.bat 执行步骤

| 步骤 | 说明 |
|------|------|
| 1 | 检查/下载 `hutb_downloader.exe` |
| 2 | 检查/下载 `dependencies` 目录 |
| 3 | 检查并终止占用端口 2000 的进程 |
| 4 | 检查/下载 `hutb` 目录 |
| 5 | 检查/解压 7zip 工具 |
| 6 | 检查/解压 miniconda3 |
| 7 | 验证 Python 虚拟环境 |
| 8 | 初始化 Python 环境并显示版本 |
| 9 | 启动 `numpy_tutorial.py` 服务 |
| 10 | 等待服务就绪（端口 3000） |
| 11 | 启动 CarlaUE4.exe 模拟器 |

## 技术实现

### 延迟变量扩展
使用 `enabledelayedexpansion` 和 `!变量!` 语法实现动态变量更新

### 时间计算
```batch
for /f "tokens=1-3 delims=:." %%a in ("%TIME%") do (
    set /a "START_H=%%a, START_M=1%%b-100, START_S=1%%c-100"
)
```

### 进度条生成
```batch
set /a "BAR_SIZE=PERCENT/2"
for /l %%i in (1,1,!BAR_SIZE!) do set "PROGRESS_BAR=!PROGRESS_BAR!#"
```

### 函数调用
- `call :show_progress "步骤说明"` - 显示进度
- `call :step_result "status" "消息"` - 显示步骤结果
- `call :show_summary` - 显示汇总信息
