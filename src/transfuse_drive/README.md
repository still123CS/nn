# TransFuser 项目简介

TransFuser 是一种基于 Transformer 的多模态传感器融合自动驾驶模型，专为 CARLA 仿真平台设计。

## 核心目标
通过跨模态注意力机制，融合相机图像与激光雷达点云信息，提升自动驾驶在复杂场景下的感知鲁棒性。

## 应用场景
雨天、夜间、多车交互等复杂仿真环境下的端到端自动驾驶控制任务。

## 核心创新点
- **跨模态 Transformer 融合机制**
  利用自注意力与交叉注意力模块，动态关联图像特征与点云特征，实现像素级与点级的信息对齐与互补。

- **端到端感知 - 决策一体化**
  模型直接从多传感器输入预测车辆控制指令（转向、油门、刹车），无需显式中间感知模块。

- **仿真环境下的鲁棒性优化**
  针对 CARLA 仿真中的噪声、光照变化、动态障碍物等场景进行了数据增强与模型微调。

## 环境依赖
- 操作系统: Ubuntu 20.04 / Windows 10+
- Python: 3.8+
