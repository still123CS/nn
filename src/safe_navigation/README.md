# 无人车安全导航

## 无人车导航简介

### 什么是无人车导航？

无人车导航是指自动驾驶汽车通过感知环境、定位自身、规划路径和控制执行，实现从起点到目的地的自主行驶技术。它是自动驾驶系统的核心组成部分。

### 无人车导航的主要模块

1. **感知**  
   - 使用摄像头、激光雷达、毫米波雷达等传感器感知周围环境  
   - 识别车道线、交通标志、障碍物、行人、其他车辆等  

2. **定位**  
   - 结合GPS、IMU、轮速计等传感器确定车辆位置  
   - 使用SLAM（同步定位与地图构建）技术在未知环境中定位  

3. **规划**  
   - 全局路径规划：从起点到终点的宏观路径  
   - 行为决策：判断何时变道、超车、停车等  
   - 局部路径规划：避障、跟车等实时轨迹规划  

4. **控制**  
   - 将规划结果转化为油门、刹车、转向等具体指令  
   - PID控制、模型预测控制（MPC）等算法

## 本项目的AirSim无人车模拟

本项目使用Microsoft AirSim模拟器，在虚幻引擎的AirSimNH环境中进行无人车基础控制演示，实现了简单的直行和路口转弯功能。

### AirSim无人车基础控制演示

#### 项目简介

这是一个基于AirSim模拟器的无人车基础控制演示程序。通过Python脚本控制虚幻引擎中的车辆模型，实现直行、减速、路口停车观察和90度转弯等功能。

#### 环境要求

- Python 3.6+  
- AirSim 1.8.1  
- 虚幻引擎 + AirSimNH环境
