1、项目说明
该程序基于 CARLA 自动驾驶仿真平台，构建了一个交通标志检测数据自动采集系统。
后续功能待扩充，目前完成的是自动驾驶的功能。
2、使用步骤
下载并安装 CARLA（推荐 0.9.11）
确保 Python 版本为 3.7（与 CARLA 0.9.11 匹配）
请把Carla模拟器部署在与项目nn同级的文件夹下，确保egg文件的路径为‘D:\software\workspace\Git\Carla\WindowsNoEditor\PythonAPI\carla\dist\carla-0.9.11-py3.7-win-amd64.egg’
本项目的路径为‘D:\software\workspace\Git\nn\src\driving_violation_detection\main.py’
运行main.py
