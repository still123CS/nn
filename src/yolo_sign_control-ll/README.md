# 🚗 CARLA 环境下的交通标志检测与车辆控制
本项目在 CARLA 仿真器中实现自动驾驶车辆行为模拟，基于实时交通标志检测。使用 YOLOv8 目标检测模型识别交通标志（如停车标志、限速标志），车辆会根据识别结果自动调整行驶行为。项目同时通过 Pygame 实时显示驾驶员视角画面。

---

## 📁 项目结构
- `Main.py` – 主程序，运行 CARLA 仿真，集成基于 YOLO 的交通标志检测与车辆控制功能。

---

## 🛠 环境依赖
请在 Python 环境中安装以下库：
```bash
pip install carla pygame numpy torch ultralytics
```

### 额外依赖
- **CARLA 仿真器**（版本 ≥ 0.9.13）：可从 [CARLA GitHub](https://github.com/carla-simulator/carla) 下载。
- **CUDA**（可选）：若有 GPU 可用于加速 YOLOv8 推理。
- **Python ≥ 3.7**

---

## 📦 使用的预训练模型
### 🧠 YOLOv8n
- **模型文件**：`yolov8n.pt`
- **来源**：由 [Ultralytics](https://github.com/ultralytics/ultralytics) 提供
- **用途**：从 RGB 相机输入中实时检测交通标志（如停车标志、限速标志）

脚本加载模型代码：
```python
model = YOLO("yolov8n.pt")
```

---

## 🎮 功能特性
- 自动驾驶车辆基于路径点实现车道保持行驶
- 对实时摄像头画面使用 YOLOv8 进行交通标志检测
- 识别停车标志并执行制动
- 根据检测到的限速标志调整车速
- 识别交通信号灯（红灯时制动）
- 在随机位置生成动态交通流与交通标志
- 通过 Pygame 实时显示画面
- 仿真运行 2 分钟后自动停止

---

## 🧪 工作原理
1. **CARLA 环境搭建**
   - 连接 CARLA 服务器
   - 生成主车、随机交通流以及交通标志（停车标志 + 限速标志）
   - 在车辆上挂载 RGB 摄像头

2. **YOLOv8 推理**
   - 摄像头每一帧图像输入 YOLOv8 模型
   - 检测交通标志的边界框与类别标签

3. **车辆控制**
   - 根据限速标志调整车速
   - 遇到停车标志时全力制动
   - 遵循交通信号灯状态行驶

4. **可视化**
   - 通过 Pygame 显示驾驶员视角画面
   - 在终端输出检测结果与车辆动作日志

---

## ▶️ 运行方法
1. **启动 CARLA 仿真器**
   ```bash
   ./CarlaUE4.sh
   ```

2. **运行 Python 脚本**
   ```bash
   python Main.py
   ```

仿真将持续运行 2 分钟并自动停止。

---

## 🧼 资源清理
仿真结束后会自动销毁所有实体（车辆、传感器、交通标志等），避免内存泄漏。

---

## 📌 注意事项
- 目前检测为基础实现，使用轻量模型 `yolov8n`。如需更高精度，可替换为 `yolov8s.pt`、`yolov8m.pt` 等模型。
- 需确保 CARLA 场景中包含脚本所引用的交通标志模型资源。
- 检测依赖 YOLO 预设类别，如需精准分类交通标志，建议重新训练模型。
