# vehicle_autonomous_core 项目说明

## 项目简介
基于 CARLA 仿真平台实现的自动驾驶功能模块，包含车辆控制、传感器采集、障碍物检测、车道保持、自动行驶等功能。

## 运行环境
- 操作系统：Windows 10/11
- Python 版本：3.10
- 仿真环境：CARLA 0.9.16
- 依赖库：carla、math、time

## 功能模块
- 车辆自动行驶与稳定控制
- 车道保持功能
- 车载摄像头视觉采集
- 雷达障碍物检测与预警
- 预设路径自动行驶

## 使用方法
1. 启动 CARLA 模拟器
2. 运行对应功能脚本
3. 视角自动跟随车辆，终端实时输出状态
